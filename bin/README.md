# gotpoint-assignment-user-notes-management

The goal is to build a RESTful API for a “notes” application using Spring Boot. By using the Use a multilayered architecture and MySql as your database to store the notes. 

This application code is divided into the following layers
1) Controller (RESTful WS)
2) ServiceAPI Layer (Service API)
3) Service Layer( Service Implementation)
4) Repository Layer
